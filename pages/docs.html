Getting Started

AuthorKit is a suite of WordPress plugins designed specifically for self-publishing authors. This guide will help you get up and running in minutes.

Prerequisites: WordPress 5.0+ and PHP 7.0+ required.

Installation

Method 1: WordPress.org (Recommended)

  1. Log in to your WordPress admin panel
  2. Navigate to Plugins → Add New
  3. Search for "AuthorKit"
  4. Click Install Now on the AuthorKit plugin
  5. Click Activate once installed

Method 2: Manual Upload

  1. Download the plugin .zip file from WordPress.org
  2. Navigate to Plugins → Add New → Upload Plugin
  3. Choose the .zip file and click Install Now
  4. Click Activate once installed

Method 3: FTP Upload

  1. Extract the .zip file
  2. Upload the authorkit folder to /wp-content/plugins/
  3. Activate the plugin in WordPress admin under Plugins

Books Module

The Books module provides complete catalog management for your published works.

Adding a Book

  1. Navigate to AuthorKit → Add New Book
  2. Enter the book title and description
  3. Add a featured image (book cover)
  4. Fill in meta fields (ISBN, price, rating, etc.)
  5. Add purchase links (Amazon, etc.)
  6. Assign categories and series
  7. Click Publish

Book Meta Fields

  • • Sort Order
  • • Coming Soon Status + Release Date
  • • Author Name + Profile URL
  • • Illustrator Name + Profile URL
  • • Star Rating (0-5)
  • • Review Count
  • • Available Formats (Kindle, Paperback, Hardcover, Board Book, Audio CD, Audiobook)
  • • ISBN, ASIN
  • • Pages, Publisher
  • • Publication Date, Language
  • • Dimensions, Weight
  • • Prices (INR, USD)
  • • Purchase Links (Amazon India, Amazon.com, Other)

Shortcodes

Display books anywhere on your site using shortcodes.

Display Single Book

[authorkit_book id="123"]

Replace 123 with your book's post ID

Display Book Grid

[authorkit_books count="6" columns="3"]

Display Books by Category

[authorkit_books category="fiction" count="4"]

FAQ

Is AuthorKit compatible with my theme?

Yes! AuthorKit is designed to work with any properly coded WordPress theme.

Will it slow down my website?

No. AuthorKit uses vanilla PHP and JavaScript with no dependencies, ensuring minimal performance impact.

Can I use it with WooCommerce?

Yes, but it's not required. AuthorKit works standalone or alongside WooCommerce.

Do I need coding skills?

No! AuthorKit is designed for non-technical users with an intuitive admin interface.

How do I get support?

Free users can use the WordPress.org support forums. Premium users get priority email support.
