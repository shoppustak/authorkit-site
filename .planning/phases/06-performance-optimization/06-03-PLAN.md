    Create an image optimization script and convert existing JPG images to WebP:

    1. Create scripts/optimize-images.sh:
    ```bash
    #!/bin/bash
    set -e

    echo "🖼️  Optimizing images..."

    # Check if cwebp is installed
    if ! command -v cwebp &> /dev/null; then
        echo "Installing webp tools..."
        if [[ "$OSTYPE" == "darwin"* ]]; then
            brew install webp
        else
            sudo apt-get update && sudo apt-get install -y webp
        fi
    fi

    # Convert JPG/PNG images to WebP
    for img in images/*.{jpg,jpeg,png}; do
        if [ -f "$img" ]; then
            output="${img%.*}.webp"
            echo "Converting $img to $output..."
            cwebp -q 85 "$img" -o "$output"

            # Create smaller variants for responsive images
            base="${img%.*}"
            ext="${img##*.}"

            # Create 320w variant
            if [[ "$OSTYPE" == "darwin"* ]]; then
                sips -Z 320 "$img" --out "${base}-320.${ext}" 2>/dev/null || true
                cwebp -q 85 "${base}-320.${ext}" -o "${base}-320.webp" 2>/dev/null || true
            fi

            # Create 640w variant
            if [[ "$OSTYPE" == "darwin"* ]]; then
                sips -Z 640 "$img" --out "${base}-640.${ext}" 2>/dev/null || true
                cwebp -q 85 "${base}-640.${ext}" -o "${base}-640.webp" 2>/dev/null || true
            fi
        fi
    done

    # Report results
    echo "📊 Image optimization complete:"
    echo "Original images:"
    ls -lh images/*.{jpg,jpeg,png} 2>/dev/null || true
    echo ""
    echo "WebP images:"
    ls -lh images/*.webp 2>/dev/null || true
    ```

    2. Make script executable: chmod +x scripts/optimize-images.sh

    3. Run the script to convert existing images:
    - tom-vining.jpg → tom-vining.webp (and responsive variants if possible)
    - sarah-book.jpg → sarah-book.webp (and responsive variants if possible)

    4. The script will:
    - Check for and install webp tools if needed
    - Convert all JPG/PNG images to WebP with 85% quality
    - Create responsive variants at 320px and 640px widths
    - Report file size savings

    Note: hero-dark.webp and hero-light.webp are already optimized.
